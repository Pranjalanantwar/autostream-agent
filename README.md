\# AutoStream Agent



\## What this does

This is a simple AI chatbot that:

\- Answers pricing questions

\- Detects when user wants to buy

\- Collects name, email, platform

\- Uses a JSON file as knowledge (RAG)



\## How to run

python main.py



\## Example

User: hi  

Agent: Hello  



User: pricing  

Agent: Shows plans  



User: buy  

Agent: Collects details
